# Stride Footwear

Stride Footwear is a premium e-commerce web application designed to bring an authentic, region-specific footwear shopping experience to the Indian market. Combining modern aesthetics with traditional styles, Stride Footwear features a specialized "Indian Collection" highlighting cultural favorites such as Juttis and Kolhapuris alongside contemporary shoe designs.

## Features

*   **Curated Collections:** Extensive product catalog including Western trends and an exclusive traditional Indian Collection.
*   **Cultural Authenticity:** Tailored content, localized reviews, and product descriptions to connect with the target demographic.
*   **Modern UI/UX:** A visually stunning and responsive web interface employing modern styling and cohesive color palettes.
*   **User Navigation:** Intuitive user flow with dedicated sections for Men's and Women's footwear, Sales, Login/Signup, and Account management.
*   **Comprehensive 'About Us':** A detailed narrative highlighting our brand's mission, our team, and the cultural inspiration driving our designs.

## Technologies Used

*   **Structure:** HTML5
*   **Styling:** Modern Vanilla CSS (for layout, animations, and typography)
*   **Functionality:** Vanilla JavaScript

## Project Structure

*   `index.html`: Main landing page
*   `html/`: Contains additional pages like Account, Login, Signup, About Us, etc.
*   `css/`: Stylesheets for UI design
*   `js/`: Scripts for interactive elements and data components

## How to Run Locally

You can run this project simply by serving the root directory using a local web server (like VS Code Live Server or python's `http.server`) and opening `index.html` in your browser.
